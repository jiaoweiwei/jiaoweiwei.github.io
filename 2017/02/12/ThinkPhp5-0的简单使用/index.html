ThinkPhp5.0的简单使用

1、引入公共文件
{include file=”public/header” /}

2、缓存cache
1）在配置文件中，添加
‘cache’ => [
‘type’ => ‘File’, //类型
‘path’ => CACHE_PATH, //缓存路径
‘prefix’ => ‘’, //缓存前缀
‘expire’ => 0, //缓存有效时间
],
支持的缓存类型包括file、memcache、wincache、sqlite、redis和xcache。
设置缓存：Cache::set(‘name’,$value,3600);
缓存自增：Cache::inc(‘name’,3); //步数为3
缓存自减：Cache::dec(‘name’,3); //步数为3
获取缓存：dump(Cache::get(‘name’));
删除缓存：Cache::rm(‘name’);
获取并删除缓存：Cache::pull(‘name’);
清空缓存：Cache::clear();

2）$options = [
// 缓存类型为File
‘type’ => ‘File’,
// 缓存有效期为永久有效
‘expire’ => 0,
// 指定缓存目录
‘path’ => APP_PATH . ‘runtime/cache/‘,
];

// 缓存初始化
// 不进行缓存初始化的话，默认使用配置文件中的缓存配置
cache($options);

// 设置缓存数据
cache(‘name’, $value, 3600);
// 获取缓存数据
var_dump(cache(‘name’));
// 删除缓存数据
cache(‘name’, NULL);

// 设置缓存的同时并且进行参数设置
cache(‘test’, $value, $options);

可以同时使用多个缓存类型

3、session
use think\Session；
session初始化
1）配置文件中添加
‘session’ => [
‘prefix’ => ‘think’,
‘type’ => ‘’,
‘auto_start’ => true,
],
2)调用init
Session::init([
‘prefix’ => ‘module’,
‘type’ => ‘’,
‘auto_start’ => true,
]);
使用【使用作用域的话，后面加上‘作用域名’】
赋值：set Session::set(‘name’,’thinkphp’);
判断是否存在：has
取值：get
删除：delete
指定作用域：Session::prefix(‘think’);
清空：clear
二维数组
Session::set(‘name.键值’,’thinkphp’);

4、cookie
cookie初始化
Cookie::init([‘prefix’=>’前缀’,’expire’=>’存储时间’,’path’=>’存储路径’]);
其他参数
// cookie 名称前缀
‘prefix’ => ‘’,
// cookie 保存时间
‘expire’ => 0,
// cookie 保存路径
‘path’ => ‘/‘,
// cookie 有效域名
‘domain’ => ‘’,
// cookie 启用安全传输
‘secure’ => false,
// httponly设置
‘httponly’ => ‘’,
// 是否使用 setcookie
‘setcookie’ => true,
简单使用：【我们在设置的时候可以指定了前缀，那我们再进行其他操作的时候，就需要指定前缀】
set设置 Cookie::set(‘name’,’value’,[‘prefix’=>’think_’,’expire’=>3600]);
has判断
get获取
delete删除
clear清空

5、模型
模型定义：
namespace app\index\model;

use think\Model;

class User extends Model
{
}

设置数据表
protected $table = ‘think_user’; //设置当前模型对应的完整数据表名称

//设置当前模型的数据库连接
protected $connection = [
// 数据库类型
‘type’ => ‘mysql’,
// 服务器地址
‘hostname’ => ‘127.0.0.1’,
// 数据库名
‘database’ => ‘thinkphp’,
// 数据库用户名
‘username’ => ‘root’,
// 数据库密码
‘password’ => ‘’,
// 数据库编码默认采用utf8
‘charset’ => ‘utf8’,
// 数据库表前缀
‘prefix’ => ‘think_’,
// 数据库调试模式
‘debug’ => false,
];

模型的调用我们可以使用静态调用，也可以实例化调用
1）静态调用
eg：$user = User::get(1);
2）实例化调用
eg：$user = new User;
3)使用Loader类实例化（单例）
eg：Loader::model(‘User’);
4)使用助手函数model
$user = model(‘User’);
上面都是手册上的内容，大概了解一下

在实例中，我们需要连接model对数据进行操作。

1）连接数据库【database.php】
在配置文件中填写数据库名称及密码

2）model中，我们需要先调用一下model类和db类，
use think\Model;
use think\Db;

新建方法
class model名称 extends Model
{
function 方法名(){
$data = Db::table(‘表名’)
->select();
return $data;

}

}
和我们之前的tp3一样，需要给在控制器中传过参数，再返回给控制器

下面说一下model层中对数据的操作
1）添加数据：insert（）
如果不使用model，那么我们可以直接执行原生的sql语句，也可以使用save方法，同时我们还可以获取到自增id
$user = new user(); //实例化
$user->name = ‘thinkphp’; //设置值
$user->email = ‘thinkphp@qq.com’;
$user->phone = ‘123456’;
$user->save(); //进行添加
echo $user->id; // 获取自增ID
注意：不要在同一个实例里面多次新增数据，如果确实需要新增，我们可以使用下面的方法。
$user = new User;
$user->name = ‘thinkphp’;
$user->email = ‘thinkphp@qq.com’;
$user->save();
$user->name = ‘onethink’;
$user->email = ‘onethink@qq.com’;
// 第二次开始必须使用下面的方式新增
$user->isUpdate(false)->save();
如果进行批量新增，我们就需要下面的方法，使用saveAll进行添加，返回值包含自增id和数组。
$user = new User;
$list = [
[‘name’=>’thinkphp’,’email’=>’thinkphp@qq.com’],
[‘name’=>’onethink’,’email’=>’onethink@qq.com’]
];
$user->saveAll($list);

2）修改
model层中修改使用update(),
eg：function upd($id,$info){
$data = Db::table(‘user’)
->where(‘id’,$id)
->update($info);
return $data;
}

如果我们不使用model，那么我们也可以使用save方法，来进行保存

3）删除
model层中删除数据使用delete()，
eg：function del($id){
$data = Db::table(‘user’) //连接表
->where(‘id’,$id) //删除条件
->delete();
return $data;
}

如果不用model，我们直接在控制器上 对数据进行操作，
有以下几种办法
【1】、根据主键删除：User::destroy(‘主键id’); //User是model的文件名
【2】、条件删除：User::destroy([‘status’ => 0]); //括号中为条件。
【3】、通过数据库类的查询条件删除：User::where(‘id’,’>’,10)->delete(); //where中为条件，执行delete方法

4）查询
model中的查询：
function select(){
$data = Db::table(‘user’)
->select();
return $data;

}

其中，查询单条语句：find（），查询多条语句：select（），条件：where（）
获取某个字段或某个列的全部值：column（）
我们还可以使用limit，order

那我们如果不使用model，只在控制器进行操作，就会使用到下面的方法

【1】、实例化模型后调用查询方法
$user = new User();
// 查询单个数据
$user->where(‘name’, ‘thinkphp’)
->find();
【2】、获取多个数据
$list = User::all(‘1,2,3’);
或
$user = new User();
// 查询数据集
$user->where(‘name’, ‘thinkphp’)
->limit(10)
->order(‘id’, ‘desc’)
->select();
【3】、获取某个字段或者某个列的值
// 获取某个列的所有值
User::where(‘status’,1)->column(‘name’);
// 以id为索引
User::where(‘status’,1)->column(‘name’,’id’);
【4】、通过Query类查询
User::where(‘id’,’>’,10)->select();
User::where(‘name’,’thinkphp’)->find();

4）控制器方面我们需要调用我们的model文件
eg：
<?php
namespace app\index\controller;
use app\index\model\user; //调用model文件夹中的user文件
class Index extends Base
{
public function index()
{
$model = new user(); //实例化model
$id = 2;
$info = array(‘name’=>’李四’,’email’=>’123123’,’phone’=>’123123’);
$model->upd($id,$info); //使用upd方法
echo “修改成功”;
}
}
